# If you are in Mainland China
Use
```sh
npm --registry="http://mirrors.cloud.tencent.com/npm/"
```
instead of `npm` in the following commands.

# Install packages
```sh
npm install
```
Or better, use `pnpm` instaead of `npm`.

# Install browser
```sh
npx playwright install firefox
```

# Run with
```sh
npm start
```
